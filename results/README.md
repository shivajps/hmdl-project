# Results

This folder should contain the **actual** output artifacts generated by
`training/evaluate.py` and related scripts — not manually authored figures.

- `tables/` — CSV/XLSX exports of Tables 2–6 as produced by your evaluation runs
- `figures/` — training curves, per-class performance plots (Figures 3–13 as manually redrawn, non-AI-generated)
- `confusion_matrix/` — per-dataset confusion matrices
- `roc/` — ROC curve data/plots per method
- `pr_curves/` — Precision-Recall curve data/plots
- `shap/` — SHAP value exports from actual model runs

**Do not populate these folders with placeholder or illustrative numbers.**
If a table/figure isn't backed by a completed run yet, leave the folder empty
or mark it `PENDING` in this README rather than committing invented data.
